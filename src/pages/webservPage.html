Welcome to Webserv

Project Overview

Webserv is a custom HTTP server written in C++, designed to provide a deep understanding of how web technologies and HTTP communication work. This project involves implementing a web server that can serve HTML pages and handle basic web functionalities.

Usage

To get started with Webserv, follow these steps:

  1. Clone the repository to your local machine:
    2. https://github.com/linhtng/Webserv.git
  2. Compile the server:
    3. make
  3. Run the server with a configuration file:
    4. ./webserv [configuration file]

Once the server is running, you can access it through your preferred web browser by navigating to the host and port specified in your configuration.
